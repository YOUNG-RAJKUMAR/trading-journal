📋 MT5 Setup — Step by Step
1
Allow WebRequest in MT5
Tools → Options → Expert Advisors
✓ Check Allow WebRequest for listed URLs
Add this URL exactly:
https://firestore.googleapis.com
2
Install the EA File
In MT5: File → Open Data Folder
Navigate to MQL5 → Experts
Copy RK_Journal_EA.mq5 into that folder
Open MetaEditor → press F7 to compile
3
Attach EA to a Chart
Drag EA onto any chart (e.g. EURUSD M1)
In the Inputs tab, set the values from the left panel
Click OK
4
Enable Algo Trading
Make sure the Algo Trading button in the MT5 toolbar is ON (green)
EA starts automatically — history imports on first run
