📖 How It Works

The Session High-Low Indicator tracks the highest and lowest prices during three major forex trading sessions:

🔵 Asia Session
Identifies volatility and range during early Asian hours
🔴 London Session
Captures European morning trading dynamics
🟠 NY Session
Tracks American session price action and breakouts
📊 Break Detection
Automatically detects when prices break session highs/lows
🎨 Customizable
Adjust colors, line width, and extension distance
⏱️ Nepal Timezone
All times synchronized to Nepal Standard Time (UTC+5:45)

🎯 Use Cases

✓ Support & Resistance: Use session highs/lows as key levels
✓ Breakout Trading: Identify when price breaks session extremes
✓ Range Trading: Trade within session boundaries
✓ Session Analysis: Compare volatility across sessions
✓ Risk Management: Set stop losses beyond session highs/lows

📋 Integration

This indicator is designed for TradingView Pine Script (v5) and HTML5 charting libraries.
